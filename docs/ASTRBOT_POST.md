# 开了玖 (KL9-RHIZOME) — AI 认知协议 + 技能书系统

## 这是什么

开了玖不是一个 AI 框架，而是一套在 LLM 之上运行的认知协议。它让 AI 同时持有两个不可调和的视角，通过"张力悬置"而非廉价综合来生成更深刻的思考。

## 最新功能：技能书吸收协议 (v1.2)

读完一本书后，KL9 可以导出"技能书"——包含概念图谱、双重视角记录和张力结构的 JSON 文件。其他人导入后，KL9 会继承你的阅读成果：碰撞概念自动分叉、近似概念智能合并、张力场局部重算。

**三源聚合评分系统 (v1.3)**：HLE(50%) + Arena Elo(25%) + Creative Writing(25%) 动态计算模型能力上限，语言偏差补偿（中文模型读中文书 +3%）。
