# 动画机器 — 术语表 Glossary

> 双语对照 · 拉马尔《The Anime Machine》核心术语
>
> 格式：英文术语 / 日文对应 / 中文译名 → 定义

---

## 核心概念 Core Concepts

### Anime Machine / アニメマシン / 动画机器
技术装置与感知模式的耦合配置。不是比喻，而是对赛璐珞动画生产中"动画摄影台-多层合成-低帧率"这一整套技术基础设施如何生产出特定观看方式的命名。借德勒兹-加塔利用法：机器 = 欲望流与技术流的连接。

### Multiplanar Image / 多平面画像 / 多平面图像
通过分离的平面层（赛璐珞片）垂直堆叠拍摄合成的图像。每层保持平面独立性，层间不融合为连续深空间。与透视深度的区别：深度来自层的离散跳跃，而非体积的连续缩小。

### Limited Animation / リミテッドアニメーション / 有限动画
每秒8-12帧、大量静止画面复用、仅活动角色局部的技术体系。拉马尔的逆转：不是经济限制的产物，而是积极选择——低帧率使间隔暴露，让运动在"之间"发生。

### Full Animation / フルアニメーション / 全动画
每秒24帧逐帧绘制、运动平滑连续的迪士尼范式。追求运动幻觉的完美封闭，隐藏帧之间的间隔。电影主义的技术载体。

### Animetic Interval / アニメ的間隔 / 动画间隔
帧之间的"之间性"——不被拍摄、不被填充的时间与空间。拉马尔的三层间隔：技术间隔（帧隙）→ 合成间隔（层隙）→ 感知间隔（观者填充）。动画的"生命"不在图像而在间隔中。

### Cinematism / シネマティズム / 电影主义
朝向或远离摄像机（纵深轴）的运动倾向。单点透视结构，导向统一视点与沉浸式主体。与电影本体论亲和。

### Animetism / アニメティズム / 动画主义
横向穿越图像平面（x-y轴）的运动倾向。多层分散结构，导向分布视点与分散主体。与动画本体论亲和。

### Sliding / スライディング / 滑动
角色在背景平面上横向移动但不融合层差异的技术。是最纯粹的动画主义表现——暴露平面分离而非隐藏它。

### Distribution of the Field / 場の分配 / 场的分布
注意力与可感性在多平面结构各层之间的分散配置。取自朗西埃"可感物的分配"(partage du sensible)，强调动画制造了一种非人类中心的视觉体制。

### Compositing Gap / 合成の隙間 / 合成间隙
层与层之间的空间-感知分离。不缝合的可见痕迹。滑动的技术条件。

### Tension Field / 緊張場 / 张力场
电影主义与动画主义之间的结构性共存空间。不是二元对立，而是每部作品内部的两极性张力。动画机器的核心动力来自维持而非消解这种张力。

## 技术术语 Technical Terms

### Animation Stand / 撮影台 / 动画摄影台
赛璐珞动画时代的垂直多层摄影装置。摄像机向下拍摄堆叠的赛璐珞片。动画机器的物质核心。

### Cel (Celluloid) / セル画 / 赛璐珞片
透明的醋酸纤维片，用于绘制动画的各个可动层。每层独立绘制，摄影台上叠放拍摄。

### Compositing / 合成 / 合成
将多层赛璐珞片在摄影台上叠放拍摄为单一画面的过程。多平面图像的物质生产。

### On-Twos / On-Threes / 二コマ打ち・三コマ打ち
每张画持续2帧（on-twos, 12fps等效）或3帧（on-threes, 8fps等效）的有限动画标准。全动画为on-ones（每张1帧, 24fps）。

### Bank System / バンクシステム / 素材库系统
有限动画的素材复用体系——预先绘制的通用动作序列（变身、合体、发射）在不同剧集中反复使用。

## 理论来源 Theoretical Sources

### Machine (Deleuze & Guattari) / 機械
不是封闭的技术物，而是异质性要素（技术、身体、社会）的耦合装配。拉马尔将动画摄影台+产业体系+感知习惯描述为一台"机器"。

### Distribution of the Sensible (Rancière) / 感性的分割
美学体制如何划分可见与不可见、可听与不可听、可感与不可感的边界。拉马尔将多平面结构重新描述为一种"场的分配"机制。

### Interval (McLaren) / 間
诺曼·麦克拉伦："动画不是运动的艺术，而是运动之间的艺术。每一帧之间发生的事情比每帧上存在的东西更重要。"拉马尔将其系统化为本体论概念。

## 分析用概念 Analytical Concepts

### 透视深度 vs 平面深度 (Perspectival Depth vs Planar Depth)
前者的深度来自消失点的连续后退（电影主义）；后者的深度来自分离平面的离散跳跃（动画主义）。

### 缝合 vs 非缝合 (Suture vs Non-Suture)
电影连续性剪辑隐藏剪切以维持沉浸（缝合）；多平面动画暴露层间分离以维持注意力的分散（非缝合）。

### 沉浸式主体 vs 分布主体 (Immersive Subject vs Distributed Subject)
电影主义产生被深度空间吞没的统一观看者；动画主义产生注意力在平面之间移动的分散观看者。

### 运动幻觉 vs 运动的感知 (Illusion of Motion vs Perception of Movement)
全动画追求将静止图像缝合为运动幻觉（隐藏间隔）；有限动画让观者在间隔中感知运动的发生（暴露间隔）。
