## 概述

《动画机器》不是一本关于"动画内容"的书——它是一本关于**动画如何作为一个技术-感知装置运作**的书。拉马尔的核心论断：**动画不是电影的亚类型，而是与电影平行的另一种移动影像体制。**

本技能书是对该著作的结构化蒸馏，包含概念图谱、术语表、核心论证链、参考文献索引，供 N9R20 框架在 DEEP 路由下激活使用。

---

## 核心概念

| 节点 | 英文 | 定义 |
|------|------|------|
| 动画机器 | Anime Machine | 技术装置与感知模式的耦合，非隐喻 |
| 多平面图像 | Multiplanar Image | 平面堆叠而非透视深度 |
| 有限动画 | Limited Animation | 经济限制→积极技术选择的反转论证 |
| 动画性间隔 | Animetical Interval | 帧间间隙作为意义产生场所 |
| 电影主义 | Cinematism | 朝向目标的弹道运动 |
| 动画主义 | Animetism | 平面间的滑动分布 |
| 场域分布 | Field Distribution | 注意力在多平面间的分配 |

完整概念图谱见 `concept-graph.json`（12 节点 / 16 边）。
